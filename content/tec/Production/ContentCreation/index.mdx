# Content Creation

This section is the operational manual for TEC Production's creative pipeline. It moves from high-level discovery to the granular details of technical handover. By following this architecture, the department maintains a standardized quality of output regardless of which officer is leading the project.

<Callout type="warn" title="Phase Gate Policy">
  A project **cannot advance** to the next phase until all deliverables from the current phase are signed off. Skipping gates is the leading cause of on-set chaos and post-production rework.
</Callout>

## Pipeline at a Glance

| Phase | Purpose | Core Question |
| :--- | :--- | :--- |
| **Pre-Pre** | Filter | Is this project viable? |
| **Pre-Production** | Blueprint | What exactly are we making? |
| **On-Site** | Execute | Are we capturing what we planned? |
| **Post-Production** | Refine | Is every frame broadcast-quality? |
| **Publication** | Release | Is it reaching the right audience? |
